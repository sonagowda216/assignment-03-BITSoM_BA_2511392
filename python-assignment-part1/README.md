# Assignment Part 1 - Python Basics & Control Flow

## Theme: Student Grade Tracker

This assignment contains all 4 tasks implemented in a single Python file: `part1_grade_tracker.py`

### Tasks Included:

#### Task 1: Data Parsing & Profile Cleaning (5 marks)
- Parses raw student data with inconsistent formatting
- Cleans names by removing whitespace and converting to Title Case
- Converts roll numbers from strings to integers
- Parses marks from comma-separated strings to integer lists
- Validates student names for alphabetic characters only
- Prints formatted profile cards for each cleaned student

**Visible Output:** Profile cards with cleaned student data

#### Task 2: Marks Analysis Using Loops & Conditionals (8 marks)
- Calculates average marks for each student
- Assigns grades based on performance thresholds (A+, A, B, C, F)
- Displays individual student statistics in a table format
- Calculates overall class statistics:
  - Class average
  - Highest and lowest scoring students
  - Pass/Fail count

**Visible Output:** Individual analysis table and class statistics summary

#### Task 3: Class Performance Summary (7 marks)
- Generates formatted performance report
- Creates structured data with name, average, and pass/fail status
- Displays performance table with proper formatting
- Computes:
  - Number of passed vs failed students
  - Class topper with average
  - Overall class average

**Visible Output:** Formatted performance report with metrics

#### Task 4: String Manipulation Utility (5 marks)
- Strips leading and trailing whitespace from essay
- Converts text to Title Case
- Counts occurrences of specific word (case-insensitive)
- Replaces words with substitutes
- Splits text into sentences with numbering
- Prints each sentence separately with proper formatting

**Visible Output:** Step-by-step string transformations

### Running the Code:

```bash
python part1_grade_tracker.py
```

All outputs are printed to console with clear section headers and formatted tables.
